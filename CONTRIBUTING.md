# Contributing to OpenSlide PPT Generator

Thank you for considering contributing!

## How Can I Contribute?

### Add New Themes

1. Create a new theme in `templates/`
2. Include: colors, typography, spacing
3. Add example presentation

### Improve Templates

- Add new slide layouts
- Improve accessibility
- Optimize for mobile

### Report Issues

- Theme rendering problems
- Missing features
- Documentation errors

## Style Guidelines

### Code Style
- TypeScript for all components
- Tailwind CSS for styling
- Semantic HTML structure

### Commit Messages
- Use present tense
- Reference issue numbers
