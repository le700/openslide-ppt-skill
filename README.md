## English

### ✨ Features

- **🤖 Agent-Native** — Built for Claude, Cursor, Codex CLI
- **🎨 Multiple Themes** — Tech, Business, Minimal, Colorful
- **⚡ Code-First** — React + Tailwind, version-controlled slides
- **📊 Rich Components** — Charts, layouts, animations
- **🎯 Brand Support** — Auto-extract brand colors & fonts

### 🚀 Quick Start

#### Installation

```bash
git clone https://github.com/le700/openslide-ppt-skill.git
cp -r openslide-ppt-skill ~/.claude/skills/openslide-ppt
```

#### Usage

```
/openslide-ppt [topic] [pages] [style]
```

**Examples:**
```
/openslide-ppt "AI Product Launch" 10 tech
/openslide-ppt "Annual Report" 15 business
/openslide-ppt "Design Portfolio" 8 minimal
```

### 🎨 Supported Themes

| Theme | Style | Best For |
|-------|-------|----------|
| `tech` | Dark, neon, geometric | Tech talks, product launches |
| `business` | Clean, professional | Reports, pitches |
| `minimal` | White space, simple | Design showcases, art |
| `colorful` | Bright, playful | Education, creative |

### 📁 Output Structure

```
slides/
├── index.tsx          # Main presentation
├── components/        # Reusable components
└── styles/           # Theme styles
```

### 💻 Example Code

```tsx
// Generated by /openslide-ppt
import { Slide } from '@open-slide/core';

export default function Presentation() {
  return (
    <>
      <Slide id="cover" className="bg-gradient-to-br from-slate-900 to-purple-900">
        <h1 className="text-7xl font-bold text-white">AI Assistant</h1>
      </Slide>
      {/* More slides... */}
    </>
  );
}
```

## 中文

### ✨ 功能特点

- **🤖 Agent 原生** — 专为 Claude、Cursor、Codex CLI 设计
- **🎨 多种主题** — 科技风、商务风、简约风、活泼风
- **⚡ 代码优先** — React + Tailwind，可版本控制的幻灯片
- **📊 丰富组件** — 图表、布局、动画效果
- **🎯 品牌支持** — 自动提取品牌色和字体

### 🚀 快速开始

#### 安装

```bash
git clone https://github.com/le700/openslide-ppt-skill.git
cp -r openslide-ppt-skill ~/.claude/skills/openslide-ppt
```

#### 使用方法

```
/openslide-ppt [主题] [页数] [风格]
```

**示例：**
```
/openslide-ppt "AI产品发布会" 10 科技风
/openslide-ppt "公司年度总结" 15 商务风
/openslide-ppt "设计作品集" 8 简约风
```

### 🎨 支持的主题

| 主题 | 风格 | 适用场景 |
|------|------|----------|
| `科技风` | 深色、霓虹、几何 | 技术分享、产品发布 |
| `商务风` | 简洁、专业 | 商业报告、路演 |
| `简约风` | 留白、极简 | 设计展示、艺术 |
| `活泼风` | 明亮、趣味 | 教育、创意内容 |

### 📊 页面模板

- **封面页** — 标题、副标题、演讲者信息
- **目录页** — 内容大纲导航
- **内容页** — 图文混排、要点展示
- **数据页** — 图表、进度条、指标
- **结尾页** — 总结、致谢、联系方式

### 🛠️ 技术栈

- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS
- **Slide Engine**: @open-slide/core
- **Export**: HTML / PDF / Images
